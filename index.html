Exploring NYC Public School Test Result Scores

In this project, I delved into the SAT performance data of NYC schools to identify key insights and trends. The analysis focused on three main areas: identifying schools with the best math results, ranking the top 10 schools based on total SAT scores, and determining which borough had the largest variation in SAT performance. Utilizing pandas for data manipulation and analysis, I was able to extract meaningful insights and present them in an easy-to-understand format. Explore the detailed findings and visualizations to learn more about the educational landscape of NYC schools.

This project involves an exploratory data analysis (EDA) of Netflix movie data from the 1990s. Using Python and Pandas, I investigated movie durations to identify the most common duration and analyzed action movies to count how many were considered "short" (less than 90 minutes).
